# Maun Smart City Website

This is a simple one-page website for Maun Smart City, a fictional real estate project.

## Structure

- `index.html`: The main HTML file containing the structure of the website.
- `style.css`: The CSS file for styling the website.
- `script.js`: The JavaScript file for adding interactivity to the website.
- `images/`: A directory containing all the images used in the website.
    - `hero-bg.jpg`: The background image for the hero section.
    - `masterplan.jpg`: The image of the masterplan.
    - `logo.png`: The logo of Maun Smart City.
- `README.txt`: This file.

## How to use

Open `index.html` in a web browser to view the website.
